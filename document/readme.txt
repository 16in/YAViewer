●概要
　MLTをはじめとした(´д｀)Editで読み込み可能なファイルを
　アスキーアートごとに表示し、(´д｀)EditやOrinrinEditor上に転送するツールです。


●動作環境
　Microsoft Windows XP 以上
　Microsoft Internet Explorer 4 以上

　なお、動作環境を満たしている場合でも動作しない場合があるかもしれません。


●基本的な使い方
　YAViewer.exeを実行してください。
　起動後の詳細はhelp/index.htmlを参照してください。

　或いはhelp/QuickStart.htmlファイルを参照してください。


●インストールの状態に戻す
　設定などを初期の状態に戻すために、次のバッチファイルを用意しました。
　必要に応じて、ディレクトリ内の該当バッチファイルをダブルクリックして実行してください。

　・delete_setting.bat
　　設定のみを初期の状態に戻します。
　　YAViewerが終了時に保存しているタブやツリーの情報は削除されません。

　・cleanup.bat
　　YAViewerが終了時に保存しているタブやツリーの情報も含めて、完全にクリーンな状態を生成します。
